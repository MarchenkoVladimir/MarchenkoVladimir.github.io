Проекты и участие

Название проекта: Bimi Boo Kids Academy
Платформа: Мобильная (iOS/Android)
Движок: Unity
Жанр: Мета-игра с коллекцией мини-игр

Описание:
▼
  • 1700+ мини-игр – разнообразные механики и жанры, динамически подгружаемые с сервера через Addressable Assets System, что позволяет экономить место на устройстве и быстро обновлять контент.
  • Гибкая система загрузки – использование Addressables для эффективного управления ресурсами и асинхронной подгрузки контента по мере необходимости.
  • Адаптивный UI – вёрстка реализована с помощью Better UI, обеспечивая корректное отображение на любых разрешениях экрана и удобный интерфейс.
  • Монетизация – реализованы внутриигровые покупки (IAP) с интеграцией Unity Purchasing, включая платный контент и виртуальные товары.
  • Аналитика и трекинг – подключена Firebase Analytics для сбора данных о поведении игроков, retention, монетизации и других метрик.
  • Оптимизация – проведена работа над снижением нагрузки на CPU/GPU, оптимизация памяти, пулинг объектов и батчинг для повышения производительности.
Технический стек:
  • Unity 2021.3.41f1
  • Addressables – динамическая загрузка контента
  • Better UI – адаптивный интерфейс
  • Firebase – аналитика и события
  • Unity IAP – система покупок
  • Оптимизация: Profiler, Asset Bundles, Object Pooling, Zenject
Результат:

Игра предлагает огромный выбор мини-игр с удобной навигацией, плавной загрузкой и стабильной работой даже на слабых устройствах. Благодаря серверной подгрузке контента, проект легко масштабируется и обновляется без необходимости выпуска новых версий.

  • Доработка UI (BetterUI) и системы монетизации (IAP)
